Muestra la fecha estimada de entrega en el checkout de WooCommerce.

== Description ==
Este plugin añade en el checkout de WooCommerce una fecha estimada de entrega configurable fácilmente. Ayuda a mejorar la experiencia de compra mostrando transparencia en los envíos.

== Usage ==
Activa el plugin y en el checkout se mostrará un mensaje con el rango de fechas estimado.
